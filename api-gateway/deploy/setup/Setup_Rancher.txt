1, Tạo Namespace cattle-system (nếu chưa có):
- CMD: kubectl create namespace cattle-system

2, Tải Rancher về:
- CMD: helm repo add rancher-latest https://releases.rancher.com/server-charts/latest --force-update
- CMD: helm repo update
- CMD: helm search repo rancher-latest/rancher --versions

3, Cài đặt Rancher:
---> Cài đặt Rancher sử dụng Helm với LoadBalancer (cho các môi trường công cộng):
- CMD: helm upgrade --install rancher rancher-latest/rancher --namespace cattle-system --set hostname=rancher.zimji.com --set replicas=1

---> Cài đặt Rancher sử dụng Helm với NodePort (cho các môi trường cục bộ):
- CMD: helm upgrade --install rancher rancher-latest/rancher --namespace cattle-system --set hostname=rancher.zimji.com --set replicas=1 --set service.type=NodePort

---> Gỡ cài đặt:
- CMD: helm uninstall rancher rancher-latest/rancher --namespace cattle-system

---> Xóa cài đặt:
- CMD: helm delete rancher --namespace cattle-system
- CMD: kubectl delete namespace cattle-system

4, Kiểm Tra Trạng Thái của Rancher:
- CMD: kubectl -n cattle-system get pods

5, Kiểm Tra Dịch Vụ Rancher:
- CMD: kubectl -n cattle-system get svc

6, Xem Logs và Tình Trạng:
- CMD: helm status rancher --namespace cattle-system

7, Áp dụng các tệp YAML:
- CMD: kubectl apply -f rancher-role.yaml
- CMD: kubectl apply -f rancher-role-binding.yaml

8, Kiểm Tra và Cập Nhật ServiceAccount:
- CMD: kubectl get serviceaccounts -n cattle-system
